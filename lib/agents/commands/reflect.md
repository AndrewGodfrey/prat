description: End-of-session self-improvement sweep
---

Review this session for behavioral mistakes worth capturing — things where a fresh Claude would have
made the same error without additional direction.

For each candidate:
1. Describe the mistake briefly
2. Identify the right file to update (agent-user.md, CLAUDE.md, modifying an existing agent or command, 
   or perhaps adding a new one)
3. Propose the specific addition or edit
4. Wait for confirmation before writing it

Skip one-time task direction ("please also include X") — only capture redirection that is likely to be relevant to 
other tasks.
